# Kuaför/Güzellik Salonu Yönetim Sistemi - ASP.NET Core MVC

## Proje Hakkında

Bu proje, kuaför ve berber salonlarının yönetimini kolaylaştıran bir web uygulamasıdır. ASP.NET Core MVC kullanılarak geliştirilen bu uygulama, salonların sunduğu işlemler, çalışanların uzmanlık alanları, uygunluk saatleri, randevu sistemi ve yapay zeka tabanlı saç modeli önerileri gibi çeşitli özellikleri kapsamaktadır. Kullanıcılar, uygulama üzerinden çalışanlarla randevu alabilir, işlemler hakkında bilgi edinebilir ve fotoğraf yükleyerek yapay zeka aracılığıyla saç kesimi ve rengi önerileri alabilirler.

Proje, REST API ile veri tabanı entegrasyonu ve LINQ sorgulamaları kullanılarak veri işlemleri gerçekleştirilmektedir.

## Teknolojiler

- **ASP.NET Core MVC** (6 ve üzeri sürümler)
- **C#**
- **PostgreSQL** / SQL Server (Veritabanı seçenekleri)
- **Entity Framework Core ORM**
- **Bootstrap 5** (Responsive Tema)
- **HTML5, CSS3, JavaScript, jQuery**
- **Yapay Zeka Entegrasyonu** (Saç modeli ve renk önerileri)

## Ana Özellikler

1. **Kuaför/Berber Tanımlamaları**: 
   - Salonlar, sundukları işlemler, süreler ve ücretler ile tanımlanabilir.
   
2. **Çalışan Yönetimi**:
   - Personel tanımlaması, uzmanlık alanları ve uygunluk saatleri yönetimi.

3. **Randevu Sistemi**:
   - Kullanıcılar, uygun çalışanlardan randevu alabilir. Randevuların onayı ve geçmiş randevu bilgileri yönetilebilir.

4. **REST API Kullanımı**:
   - Projeye ait veri tabanı işlemleri için REST API kullanılacaktır.

5. **Yapay Zeka Entegrasyonu**:
   - Kullanıcılar fotoğraf yükleyerek yapay zeka aracılığıyla saç modeli ve rengi önerileri alabilir.

## Kullanıcı Rolleri

- **Admin**: Tüm salonlar ve çalışanlar üzerinde yönetim yetkisi.
- **Employee**: Randevularını yönetip, çalışacak personeller.
- **Kullanıcı**: Salonlarda işlem seçerek randevu alabilen, saç modeli önerisi alabilen kullanıcı.
